# Vendor-Perfomance-Analysis
Advanced Vendor Performance Analysis (End-to-End)
Tools: Power BI, Python (Pandas, Matplotlib, Seaborn), SQLite

Executed an end-to-end data analytics project for Vendor Performance Analysis, processing sales, purchase, and inventory data to identify high-impact opportunities.

Performed comprehensive Exploratory Data Analysis (EDA) and data cleaning in Python using Pandas, incorporating real-time data ingestion and logging for auditability.

Solved critical business problems by executing complex SQL queries (SQLite) within the Python environment to derive key vendor and brand metrics.

Visualized trends (Matplotlib/Seaborn) and designed a professional Power BI Dashboard (Total Sales $441M, Gross Profit $134M) to track Vendor Contribution, Stock Turnover, and Low Performing Brands.

Successfully delivered actionable insights into profit margin and unsold capital, directly supporting procurement and inventory optimization strategies.
